# Data

This folder contains temporary frontend data.

## Why does this exist?

Before you have a backend or database, you need some data to practice rendering.

Later, this data should move to PostgreSQL and be returned by your Express API.

## Assignment

1. Add more fake projects.
2. Add more fake experience entries.
3. Render them in the frontend.
4. Later, delete or stop using these files once the backend works.
