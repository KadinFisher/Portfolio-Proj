# Pages

Page files contain JavaScript that only belongs to one page.

Examples:

- `home.js` runs only on `index.html`
- `projects.js` runs only on `projects.html`
- `experience.js` runs only on `experience.html`
- `contact.js` runs only on `contact.html`

## Why not put everything in `main.js`?

Because `main.js` runs on every page.

If you put all code in `main.js`, you will constantly need to check whether page-specific elements exist.

## Assignment

Implement each page one at a time. Start with the page that has the simplest feature.

Recommended order:

1. `home.js`
2. `projects.js`
3. `experience.js`
4. `contact.js`
