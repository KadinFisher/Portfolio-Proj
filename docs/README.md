## How to use these docs

You do not have to follow them perfectly linearly. Each file explains a concept and gives assignments.

The most important habit is to connect every feature across the stack:

```txt
HTML structure -> CSS styling -> JavaScript behavior -> API -> Database
```

## What to write in your own notes
